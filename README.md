## 项目介绍

WebWindow是一款基于浏览器的轻量级虚拟桌面系统，实现了类Ubuntu的桌面端多窗口体验和原生Android风格的移动端自适应体验，无需安装客户端，打开浏览器即可使用。

### ✨ 核心亮点
- 📱 **跨设备自适应**：自动识别设备类型，PC端显示多窗口桌面，移动端显示全屏安卓风格界面
- ⚡ **极致轻量**：前端基于原生JS+TailwindCSS，无重型框架，加载速度快，后端基于FastAPI，资源占用低
- 🛠️ **功能完整**：内置文件管理器、代码编辑器、终端模拟器三大核心功能
- 🔒 **安全可靠**：文件访问限制在用户主目录，防止路径遍历攻击
- 🎨 **原生体验**：支持窗口拖拽、缩放、最小化/最大化，移动端支持手势滑动、触控优化

### 📋 功能列表
| 功能分类 | 详细功能 |
|---------|---------|
| **桌面系统** | 多窗口管理、拖拽缩放、最小化/最大化/关闭、自动置顶、任务栏显示 |
| **文件管理器** | 主目录访问、目录导航、面包屑、右键/长按菜单、文件打开、列表/网格自动切换、移动端优化 |
| **代码编辑器** | Monaco编辑器集成、20+语言语法高亮、Markdown编辑/预览切换、行号显示、自动换行、工作区模式、左侧文件树 |
| **终端模拟器** | WebSocket实时通信、完整Bash支持、自适应大小、快捷键支持、彩色输出 |
| **移动端适配** | 全屏安卓体验、顶部状态栏、底部三键导航（返回/主页/概览）、横/竖屏自动适配、触控大按钮优化、列表模式浏览文件 |

## 安装部署方法

### 📦 环境要求
- Python 3.8+
- Linux/macOS/Windows系统
- 浏览器支持：Chrome/Edge/Firefox/Safari

### 🔧 安装步骤
1. **克隆项目**
```bash
git clone https://github.com/Autotos/WebWindow.git
cd WebWindow
```

2. **创建虚拟环境（推荐）**
```bash
# Linux/macOS
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **开放端口（如果需要外部访问）**
```bash
# Linux（防火墙开放5555端口）
sudo ufw allow 5555/tcp

# 云服务器需要在安全组开放5555端口入站规则
```

5. **启动服务**
```bash
# 本地运行（仅本机访问）
python main.py

# 局域网/公网部署（允许所有IP访问）
python main.py --host 0.0.0.0 --port 5555
```

6. **访问使用**
打开浏览器访问：`http://本机IP:5555`
- 本地访问：`http://127.0.0.1:5555`
- 手机访问：确保和电脑在同一个局域网，访问电脑IP+5555端口即可

## 使用方法

### 💻 PC端使用
1. **打开应用**：双击桌面图标即可打开对应应用（文件管理器/编辑器/终端）
2. **窗口操作**：
   - 拖拽标题栏移动窗口
   - 拖拽窗口边缘调整大小
   - 右上角按钮：最小化/最大化/关闭窗口
   - 点击窗口自动置顶
3. **文件管理器**：
   - 双击目录进入，双击文件打开
   - 右键点击文件/文件夹弹出菜单
   - 面包屑导航快速跳转路径
4. **代码编辑器**：
   - 支持打开单个文件或者文件夹作为工作区
   - 左侧文件树点击直接打开文件
   - 视图菜单可切换Markdown预览/编辑模式、行号显示、自动换行
   - Ctrl+S保存文件
5. **终端模拟器**：
   - 直接输入Bash命令执行，支持tab补全、Ctrl+C终止
   - 自适应窗口大小，支持彩色输出

### 📱 移动端使用
1. **布局自适应**：打开页面自动识别为移动端安卓风格，全屏显示
2. **底部导航**：
   - 返回键：关闭当前打开的窗口
   - 主页键：最小化所有窗口返回桌面
   - 概览键：移除，避免遮挡导航
3. **文件操作**：
   - 竖屏自动切换为列表模式，上下滑动浏览
   - 横屏自动切换为网格模式，充分利用宽度
   - 长按文件/文件夹弹出操作菜单
4. **编辑器适配**：自动放大字体、开启自动换行，支持触控滑动浏览代码，支持双指缩放

## 常见问题
### Q：移动端访问页面显示不全？
A：确保手机和电脑在同一个局域网，关闭VPN或者代理，使用电脑内网IP访问。

### Q：终端无法输入？
A：终端需要WebSocket连接，确保浏览器没有禁用WebSocket，防火墙没有拦截WS协议。

### Q：Markdown预览不显示？
A：强制刷新页面（Ctrl+Shift+R）清空缓存重新加载即可。
